Breadcrumbs

Description

This is a page that shows breadcrumbs. Breadcrumbs are one of, if the only, component that sits outside of a module.

Example

  • Home
  • Library
  • Data

Accessibility

Must include active and focus styling. Could role="breadcrumbs" be useful?

Markup:HTML

<ul class="breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Library</a></li>
  <li class="active">Data</li>
</ul>

<div class="row">
  <div class="col-md-12">
    <div class="module">
      ...
    </div> 
  </div> 
</div>

ol.breadcrumb,
ul.breadcrumb {
	margin-bottom: 20px;
	margin-top: 0;
	margin-left: 10px;

	background-color: transparent;
	border-radius: 0;
	padding: 0;
	a:link,

		font-size: 10px;
		font-weight: 300;

	}

		font-size: 10px;
		font-weight: 700;

	}

}
