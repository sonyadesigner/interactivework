Icons

Markup

Our icons are a custom font built using icomoon.io. As long as the font files are loaded properly, each icon can be controlled using any standard CSS font style. Each icon class can be added to any HTML element, but for consistency, please use <i> tags. The most common exception to using <i> tags will be for clickable icons. In those cases, use <a class="icon-check"></a>.

Placing text inside the i or a tags, will insert that text after the icon because the icons are applied to the :before pseudo-class. If the need arises for Tk20 to place icons after text, we will create an appropriate stylesheet.
