Description

All application content, with nearly no exceptions, will be housed in this style of module block. If the module contains a submenu (shown here at the top right - "delete, save, edit, etc"), the editing-options div must be included inside the heading div. Primary list items in the editing-options menu will collapse last, while the secondary list items collapse into a dropdown first.
