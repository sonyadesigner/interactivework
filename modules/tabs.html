Basic Usage

Tabbed pages such as this should ONLY be placed full width rows, i.e. <div class='col-md-12 col-sm-12'>. If the tabs perform a line-wrap, the functionality becomes confusing. Please attempt to give shorter tab names for this same reason.

To have a tab displayed by default, give the associated li tag a class of active and the div containing the content classes of active in.

Markup

<ul class="nav nav-tabs">
  <li class="active"><a href="#tab-page-one" data-toggle="tab" tab-index="1">Basic Usage</a></li>
  <li><a href="#tab-page-two" data-toggle="tab" tab-index="2">Markup</a></li>
  <li><a href="#tab-page-three" data-toggle="tab" tab-index="3">Tab 3</a></li>
  <li><a href="#tab-page-four" data-toggle="tab" tab-index="4">Tab 4</a></li>
</ul>
<div class="tab-content">
  <div class="tab-pane fade active in" id="tab-page-one">
    <h2>Basic Usage</h2>
    <p>Tab One Content Goes Here</p>
  </div>
  <div class="tab-pane fade" id="tab-page-two">
