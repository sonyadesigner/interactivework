Popovers and Modal Windows

Modal Window (Popup window)

Modal in a link

Modal in a button

Markup for a modal window trigger. Can be placed on nearly any element, not just anchor tags.

<a href="#" class="btn" data-toggle="modal" data-target="#example-modal">Modal</a>

This markup below is not visible when the page loads and should be placed after all visible content. If there are form elements present in the modal, place the autofocus attribute on the first one. This will receive user focus when the modal is opened.

<!-- modal -->
<div class="modal fade" id="example-modal" tab-index="-1" role="dialog" aria-labelledby="example-modallabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <div class="close" type="button" data-dismiss="modal" aria-hidden="true"><i class="icon-close"></i></div>
        <h4>Modal Title</h4>
      </div>
      <div class="modal-body">
        <input type="text" autofocus="autofocus">
        <hr>
        <input type="text">
        <hr>
        <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Voluptatem, ea, beatae, quia consequatur eius reiciendis assumenda in magnam recusandae aspernatur rerum tempore quaerat at incidunt saepe a explicabo ut natus.</p>
      </div>
      <div class="modal-footer">
        <button class="btn btn-default" data-dismiss="modal">Close</button>
        <button class="btn btn-submit" type="submit">Submit</button>
      </div>
    </div>
  </div>
</div><!-- /modal -->

Click

For more complex popovers, include the data-popovercontent attribute with a value that corresponds to the id of a hidden div containing the HTML you wish to place in the popover. This is similar to the modal markup and functionality.

<button data-popovercontent="#popover-form-1" class="btn btn-default tip" data-toggle="popover" data-trigger="click">Click</button>

<div id="popover-form-1" class="hidden">
  <form action="" class="form-horizontal">
  <div class="row">
  <div class="col-sm-3">
  <label for="name">Name: </label>
  </div>
  <div class="col-sm-9">
  <input type="text">
  </div>
  </div>
  <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Molestias, earum voluptatum rerum saepe labore reprehenderit fugit quis animi dicta maxime. Dicta fuga voluptas fugiat tenetur dolorum eaque ipsum maiores ipsam.</p>
  <button type="submit" class="btn btn-submit">Submit</button>
  </form>
  </div>
