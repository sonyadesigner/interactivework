Wizard / Progress Steps

These progress arrows are built using background images, but are dynamic in width. Inside each step are two paragraph tags, the first is the step number which is only visible at mobile screen sizes, while the second paragraph is the full title of the step and will display only on wider screens.

Each block of code for each step can be repeated as needed, but be careful that the correct classes are being used for previous/next steps. These are dependent on classes used, not where they appear in the html.

<div class="wizard-steps">

  <!-- previous steps - outlined in green -->
  <a href="#" class="step previous">
    <span class="prev-left"></span>
      <p class="step-number">1</p>
      <p class="step-title">Step 1: Enter First Information</p>
    <span class="prev-right"></span>
  </a>
  
  <!-- current step - green fill -->
  <a href="#" class="step current">
    <span class="current-left"></span>
      <p class="step-number">2</p>
      <p class="step-title">Step 2: Enter Second Information</p>
    <span class="current-right"></span>
  </a>
  
  <!-- next steps - outlined in gray -->
  <a href="#" class="step next">
    <span class="next-left"></span>
      <p class="step-number">3</p>
      <p class="step-title">Step 3: Enter Final Information</p>
    <span class="next-right"></span>
  </a>

</div>
