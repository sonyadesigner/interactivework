File Tree Structure

Description

File Tree Lines/Files/Links are styled based on the level they are nested. First tier is bold, second tier is normal paragraph style text, and third, fourth, etc tiers are normal italicized text.

Example

Expand All Collapse All

Accessibility

Not really sure how to make these accessible.
