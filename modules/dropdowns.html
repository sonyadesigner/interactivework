Native Select Menu Dropdown

Description

The down arrow icon is programmatically inserted via javascript. The custom select element will fill the full width of its containing element. It will typically follow the same layout pattern as form inputs and textareas.

Example

Accessibility

Markup:HTML


<div class="form-group">
	<div class="custom-select">
		<select name="select" id="select" class="form-control">
			<option value="null">Please Select</option>
			<option value="a1">Option One</option>
			<option value="a2">Option Two</option>
			<option value="a3">Option Three</option>
		</select>
	</div>
</div>

Custom Select Drop Down

Description

Design requirements will result in development (both front and back end) needing to "fake it" when it comes to the custom dropdowns because we are not using native form elements.

Example

  • Dropdown Item 1
  • Dropdown Item 2
  • Dropdown Item 3
  • Dropdown Item 4
  • Dropdown Item 5
  • Dropdown Item 777777777

Accessibility

Markup:HTML


<div class="form-group">						
	<div class="dropdown">
		<button class="btn dropdown-toggle" type="button" id="  sample-dropdown" data-toggle="dropdown">
			<span class="selection">Please Select</span>
		</button>
		<ul class="dropdown-menu" role="menu" aria-labelledby="  sample-dropdown">
			<li role="list item"><a href="#">Dropdown Item 1</a></li>
			<li role="list item"><a href="#">Dropdown Item 2</a></li>
			<li role="list item"><a href="#">Dropdown Item 3</a></li>
			<li role="list item"><a href="#">Dropdown Item 4</a></li>
			<li role="list item"><a href="#">Dropdown Item 5</a></li>
			<li role="list item"><a href="#">Dropdown Item 777777777</a></li>
		</ul>
	</div>
</div>
