Fluid Grid

Markup

Our new UI utilizes Bootstrap's 12-column fluid grid. This means that each row is divided into 12 equal columns, allowing us to create consistent, yet dynamic layouts.

Each row is wrapped in <div class="row"></div>. Set the width using <div class="col-md-12"> where the '12' can be anything between 1 and 12. Each row will be made up of these columns adding up to 12.

Our columns have a 0 pixel gutter, unlike the default Bootstrap grid. To create some separation for inner-module layouts, a 10 pixel padding is applied to the second column of every row.

Responsive Grid

For the most part, we will use the medium column grid. This affects the grid when the screen width is above a certain breakpoint. At or below that breakpoint each column will collapse to become a full width row.

Medium columns collapse at 960px. The 10px left padding is then removed so that all columns line up properly.
